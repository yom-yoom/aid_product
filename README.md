# Система адресной доставки продуктовых наборов

Минимальное веб-приложение на Spring Boot + Thymeleaf + JPA.

## Что реализовано

- каталог рационов;
- оформление заявки получателем помощи;
- просмотр и обработка заявок администратором;
- назначение доставки волонтёру;
- список доставок для волонтёра;
- переключение профилей без сложной авторизации;
- H2 по умолчанию и отдельный профиль для PostgreSQL.

## Профили интерфейса

- Получатель помощи
- Администратор
- Волонтёр

Профиль можно переключать в шапке сайта.

## Быстрый запуск

```bash
mvn spring-boot:run
```

Открыть в браузере:

- главная: `http://localhost:8080/`
- консоль H2: `http://localhost:8080/h2-console`

Параметры H2:

- JDBC URL: `jdbc:h2:mem:foodaid`
- user: `sa`
- password: пусто

## Профиль PostgreSQL

Для запуска с PostgreSQL:

```bash
mvn spring-boot:run -Dspring-boot.run.profiles=postgres
```

Перед запуском заполни настройки в `src/main/resources/application-postgres.yml`.

## Основные маршруты

- `/` — главная страница
- `/rations` — каталог рационов
- `/requests/new` — форма оформления заявки
- `/admin/requests` — список заявок администратора
- `/admin/rations/new` — добавление рациона
- `/volunteer/deliveries` — доставки волонтёра
